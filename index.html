Aurora Shao - Portfolio Design System

Reliable, Friendly, Calm, Fresh

This website includes the overall style of Aurora Shao's portfolio website, including the logo, color palette, typography, page layouts, and major components. The overall style aims to appropriately reflect my four personal branding attributes as shown above.

This design system is created based on the Template provided by Professor Cynthia Putnam and Bootstrap 4 framework.

Foundations

The Foundations include the color palette, typographical hierarchy, as well as my logo with their variations for different usages.

Components

The Components include the navigation, footer, and buttons used in the website. Each component is shown with a responsive example, accessibility guidelines, usage practices, and the code.

Page Patterns

The Page Patterns include the layouts of Homepage and Projects Page of the portfolio website. Layouts are shown with multiple screen sizes and detailed dimensions.
