Navigation

The Nagivation Bar is an essential component in the design system. It sticks at the top through the website and is responsive to screen size.

Example

The example is responsive. Adjust the web browser window to demonstrate the responsive behaviors.

    AURORA SHAO
  • WORK
  • ABOUT
  • RESUME

Accessibility

Navigation is the way users find and access the different pages available. Therefore, it is imperative that navigation is accessible.

  • Put the links within an unordered list structure so that a screen reader will read out how many things are in the list to give visually impaired users the most information possible about the contents of the navigation.
  • Use the semantic <nav> tag around the unordered list and links with an ARIA role='navigation' attribute.
  • Optionally, attach an aria-label attribute to navigation to give users of assistive technology as much information as possible.

Usage and Best Practices

The navigation sticks at the top through the website for users to access different pages.

  • Make sure the navigation is responsive.
  • From the left to the right, order the nagivation items based on their importance.
  • Include a shadow to differentiate from main content.
  • Link the icon and name to the home page.
