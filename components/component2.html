Footer

Footer provides additional social media information. It is placed at the bottom through the website and is responsive to screen size.

Example

The example is responsive. Adjust the web browser window to demonstrate the responsive behaviors.

Accessibility

Footer usually includes the information repeated at the end of every page of a website.

  • Put the links within an unordered list structure so that a screen reader will read out how many things are in the list to give visually impaired users the most information possible about the contents of the navigation.
  • Use the semantic <footer> tag around the unordered list and links with an ARIA role='contentinfo' attribute.
  • Optionally, attach an aria-label attribute to footer to give users of assistive technology as much information as possible.

Usage

Footers stick at the bottom of each page across the website.

  • Use a dark background to differentiate the footer from the main content.
  • Make sure the footer sticks at the bottom regardless of the amount of the content.
  • Include a shadow to indicate each icon is an active one.

Code


        <footer class="footer mt-auto">
          <div class="container">
            <div class="d-flex justify-content-center">
              <div class="footer-item align-middle"><img src="../img/linkedin.png class="shadow" alt="LinkedIn Icon></div>
              <div class="footer-item align-middle"><img src="../img/ins.png class="shadow" alt="Instagram Icon"></div>
              <div class="footer-item align-middle"><img src="../img/dribbble.png class="shadow" alt="Dribbble Icon"></div>
            </div>
          </div>
        </footer>
