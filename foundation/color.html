Color

The system uses the starry-night blue as brand color and Turquoise as highlight color to express the brand attributes. These two colors are balanced by minimalist gray of different levels of darkness.

White
Color Hex Code: #FFFFFF

Usage

Website background, Button Text Hover
Base 2

Color Hex Code: #E9E9E9

Usage

Decorative Elements, Image Placeholder
Base 3

Dark Gray
Color Hex Code: #9D9D9D

Usage

Navigation Links, Project Description
